AOT (Ahead of time)
1. Write a Java Program.
2. Converting in machine language.
3. Faster.
4. Not Platform independent (C, C++, C# without .net core.)


JIT (Just in time)
1. Write a Program.
2. Run on machine.
3. Compiled just before running.
4. Java, JS, Go, C#
5. Java Runtime. JRE, .NET, NodeJS.
